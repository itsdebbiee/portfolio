Simply Desserts is a website that I created along with two other partners, Paula Le and Michelle Wang as an assignment for a Usability and Information Architecture class to build an informational website about a topic of our choice. As all of us have a sweet tooth and love discovering new places to eat, so we though, 'Why not create a website that we would find useful ourselves, too?'

To create this website, we first had to interview different types of users to find out what they would want on the website. We found that there were 3 categories: Dessert Snobs, Tourists/ Points of Interest, and At-Home Bakers. We consolidated the information after making user flows and user scenarios and the end product is this website - Simply Desserts! Be sure to check it out if you want to find information on desserts in general, discover new dessert businesses in San Diego, or if you want to create desserts in the comfort of your own kitchen.
